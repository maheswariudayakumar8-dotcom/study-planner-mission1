# Study Planner App - Mission 1

## Problem Statement
Students often forget assignments and struggle to manage their study schedules.

## Research
- Students need reminders for tasks
- Simple dashboard is required
- Exam tracking is important

## User Flow
Login → Dashboard → Add Task → Save Task → View Tasks

## Wireframes
Created using Figma (3 screens)
- Login Screen
- Dashboard Screen
- Add Task Screen
